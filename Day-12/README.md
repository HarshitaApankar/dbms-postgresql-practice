# Employee Management System Database

## 📌 Overview

This project is a **PostgreSQL Employee Management System** that manages departments, employees, projects, and salary records. It demonstrates database design, relationships, data insertion, and SQL queries for retrieving and analyzing information.

## 🛠️ Technologies Used

* PostgreSQL
* SQL

## 📂 Tables

* **Departments** – Stores department details.
* **Employees** – Stores employee information and manager hierarchy.
* **Projects** – Stores department project details.
* **Salarys** – Stores employee salary records.

## ✨ Features

* Table creation with Primary and Foreign Keys
* Sample data insertion
* Filtering and sorting
* Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY and HAVING
* INNER JOIN, LEFT JOIN, and SELF JOIN
* Subqueries

## 🚀 How to Run

1. Create a PostgreSQL database.
2. Run the SQL script.
3. Execute the queries to view the results.
