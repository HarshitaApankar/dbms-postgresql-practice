# Item-Supplier Database

This DBMS-II project is based on an Item-Supplier database using PostgreSQL. It includes three tables: `Items`, `Suppliers`, and `Items_Suppliers` to represent a many-to-many relationship between items and suppliers.

## Features
- Create tables with primary and foreign keys
- Insert sample records
- Cursor to display item names based on rate
- Trigger to prevent rate less than 50
- Procedure to find the maximum of three numbers

## Technologies Used
- SQL
- PostgreSQL
- PL/pgSQL

 # Car-Driver Database

This DBMS-II project is based on a Car-Driver database using PostgreSQL. It includes `Car`, `Driver`, and `Car_Driver` tables to represent a many-to-many relationship between cars and drivers.

## Features
- Create tables with primary and foreign keys
- Insert sample records
- Cursor to display car details by driver name
- Trigger to validate driver age
- Procedure to find the sum of first 100 numbers

## Technologies Used
- SQL
- PostgreSQL
- PL/pgSQL
